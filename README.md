## 🇹🇭 ภาษาไทย (Thai Description)

**Napa AI Translate Kit** คือโปรแกรมแปลภาษาที่ออกแบบมาเพื่อความยืดหยุ่นสูงสุด โดยมาสเตอร์สามารถเลือกแปลได้ทั้งแบบประโยคต่อประโยค หรือการแปลยกไฟล์ขนาดใหญ่ พร้อมระบบ **"คัมภีร์นภา" (Translation Memory)** ที่ช่วยจดจำคำศัพท์เฉพาะทางของมาสเตอร์

### 🌟 คุณสมบัติเด่น (Features)
* **Dual Engine:** เลือกใช้ได้ทั้ง **Online (Google)** เพื่อความสละสลวย หรือ **Offline (Argos)** เพื่อความปลอดภัยของข้อมูล
* **Manual Mode:** แปลทีละคำหรือประโยค พร้อมระบบเมนูคลิกขวา (Cut/Copy/Paste) ที่สมบูรณ์แบบ
* **File Mode:** รองรับการโหลดไฟล์ `.txt` เพื่อแปลทั้งชุดข้อมูล โดยสามารถกำหนดตัวคั่น (Delimiter) ได้เอง
* **Napa Memory:** ระบบจดจำการแปลส่วนตัว บันทึกลงฐานข้อมูล JSON เพื่อเรียกใช้ซ้ำได้ทันที
* **Portable (Fast Launch):** ตัวโปรแกรมออกแบบมาให้เปิดทำงานได้รวดเร็วผ่านระบบ Directory Mode

## 🇺🇸 English (Description)

**Napa AI Translate Kit** is a versatile translation tool designed for both speed and privacy. It allows users to translate individual sentences or process entire text files with ease.

### 🌟 Key Features
* **Dual Translation Engines:** Switch between **Google Online** for best results and **Argos Offline** for data privacy.
* **Batch File Processing:** Load and translate large `.txt` files with custom delimiters.
* **Translation Memory (Napa Brain):** Built-in system to save and remember your custom translations in `napa_memory.json`.
* **User-Friendly UI:** Modern Dark Mode interface with full right-click support for productivity.
